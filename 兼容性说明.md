# 📱 手机兼容性问题诊断与解决方案

## 🔍 常见问题

### 为什么有的手机能打开，有的打不开？

可能的原因：

1. **浏览器版本过旧**
   - iOS Safari < 11
   - Android Chrome < 60
   - 微信内置浏览器版本过旧

2. **网络问题**
   - CDN 资源加载失败（jsDelivr、Google Fonts）
   - 网络不稳定导致脚本加载超时
   -部分地区需要科学上网才能访问某些CDN

3. **设备性能**
   - 老旧手机 CPU 性能不足
   - 内存不够，导致 3D 动画卡顿
   - GPU 渲染能力弱

4. **系统版本**
   - iOS < 11
   - Android < 6.0
   - 系统缺少必要的 Web API 支持

## ✅ 已实施的兼容性增强

### 1. **兼容性补丁脚本** (`js/compatibility.js`)

添加了以下 polyfill：
- `String.prototype.startsWith()`
- `String.prototype.includes()`
- `Array.prototype.find()`
- `Array.prototype.filter()`
- `requestAnimationFrame`
- `fetch` API
- `Promise` 对象

### 2. **动态资源加载**

```javascript
// 关键脚本动态加载，失败时不影响主功能
var script = document.createElement('script');
script.src = '...';
script.onerror = function() {
    console.warn('脚本加载失败，使用降级方案');
};
document.head.appendChild(script);
```

### 3. **安全的 localStorage 访问**

```javascript
window.safeLocalStorage = {
    getItem: function(key) {
        try {
            return localStorage.getItem(key);
        } catch (e) {
            return null; // 无痕模式或禁用 localStorage
        }
    }
};
```

### 4. **错误处理和降级方案**

- 脚本加载失败时显示友好提示
- 3D 动画不支持时降级到 2D
- 音效不支持时静默失败
- Canvas 不支持时使用图片替代

## 🔧 针对不同问题的解决方案

### 问题 1: 页面完全打不开

**可能原因**：
- CDN 资源加载失败
- 网络连接问题

**解决方案**：
1. 刷新页面（下拉刷新）
2. 检查网络连接
3. 切换到更好的网络（Wi-Fi vs 4G）
4. 清除浏览器缓存

### 问题 2: 页面打开了但骰子不动

**可能原因**：
- 不支持 CSS 3D Transform
- 设备性能不足

**解决方案**：
1. 更新到最新版本浏览器
2. 关闭其他应用释放内存
3. 使用微信"在浏览器打开"

### 问题 3: 没有音效

**可能原因**：
- 不支持 Web Audio API
- 系统静音

**解决方案**：
1. 检查系统音量
2. 更新浏览器版本
3. 不影响游戏，可继续使用

### 问题 4: 分享功能不可用

**可能原因**：
- Canvas API 不支持
- html2canvas 加载失败

**解决方案**：
1. 手动截图分享
2. 更新到 iOS 13+ 或 Android 8+
3. 使用链接分享而非图片

### 问题 5: 数据统计失败

**可能原因**：
- localStorage 被禁用（无痕模式）
- 网络问题导致 Supabase 连接失败

**解决方案**：
1. 退出无痕模式
2. 检查网络连接
3. 不影响游戏，可继续使用

## 📱 各平台测试建议

### iOS 设备

**推荐测试版本**：
- ✅ iPhone 8+ (iOS 14+)
- ⚠️ iPhone 7 (iOS 13)
- ❌ iPhone 6 及以下

**浏览器选择**：
1. Safari (最佳)
2. Chrome (良好)
3. 微信内置浏览器 (可能有限制)

### Android 设备

**推荐测试版本**：
- ✅ Android 10+ (Chrome 90+)
- ⚠️ Android 8-9 (Chrome 70-89)
- ❌ Android 7 及以下

**浏览器选择**：
1. Chrome (最佳)
2. 微信内置浏览器 (良好)
3. UC 浏览器 (可能有限制)

### 微信内置浏览器

**已知限制**：
- 部分版本不支持 async/await
- CDN 访问可能较慢
- 分享功能有限制

**建议**：
- 在微信中使用"在浏览器打开"
- 或直接用 Safari/Chrome 打开
